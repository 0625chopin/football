## 1. 경쟁 구도 정의

### 1.1 계층 구조

```
[대체재]  실축구 중계 시청 / FPL 등 판타지 스포츠
    │
[간접 C]  방치형·자동 진행 게임 (오프라인 진행, 안 봐도 돌아감)
    │
[간접 B]  실축구 배팅 (사용자의 지갑·시간을 직접 경합)
    │
[간접 A]  축구 매니저·시뮬레이션 (지속 세계·성장·이적을 공유)
    │
[직접]    자동 진행 가상 축구 리그 + 배팅
          ├─ B2B 가상 스포츠 공급사 (Betradar / Golden Race / Kiron / Inspired)
          ├─ Sportradar Simulated Reality League (SRL)
          └─ eSoccer / ESportsBattle (사람이 조작하지만 24/7 축구 배팅 상품)
```

### 1.2 계층별 정의와 football4와의 거리

| 계층 | 정의 | 대표 서비스 | football4와의 **공통점** | football4와의 **결정적 차이** |
|---|---|---|---|---|
| **직접** | 자동으로 돌아가는 가상 축구 경기를 사용자가 관전하며 베팅 | Betradar Virtual Football League, Golden Race Virtual Football, Kiron GOAL Premier, Sportradar SRL | 24/7 자동 진행 / 관전 + 배팅 / 사용자 조작 없음 / 다수 마켓 | **지속 세계가 없다.** 선수는 성장·부상·이적·은퇴하지 않고, 시즌 간 연속성이 없다. 전부 **실화폐**이며 대부분 **B2B 화이트라벨** |
| **간접 A** | 사용자가 감독이 되어 팀을 운영하는 시뮬레이션 | Hattrick, ManagerZone, Sokker, Top Eleven, OSM, Soccer Manager, Football Manager 26, ZenGM | 지속 세계 / 선수 성장·이적·유소년 / 리그·컵 구조 / 통계 깊이 | **사용자가 개입한다.** 개입이 재미의 본체이며, 배팅이 없다 |
| **간접 B** | 실축구 대상 배팅 | bet365, bwin, (국내) 베트맨 | 배팅 마켓 구조·배당 표기·정산 UX가 사실상 동일 | 대상이 **실존 경기**. 뉴스·부상 정보 등 외부 정보 우위가 존재. 국내는 **베트맨 외 전부 불법** |
| **간접 C** | 사용자가 보지 않아도 진행되는 게임 | 방치형/증분(idle/incremental) 장르 전반 | "안 봐도 세계가 돌아간다" / 짧은 재방문 루프 | 방치형은 **진행이 사용자 소유 자원의 성장**이다. football4는 사용자가 소유한 것이 없다(2차 이후 포인트뿐) |
| **대체재** | 같은 시간·주의를 소비 | 실축구 중계, FPL(2025/26 시즌 950만 매니저) | 축구 소비 시간대를 공유 | 무료·대규모 커뮤니티·실존 선수 서사 |

> **주의 — 카테고리 혼동 리스크**: football4는 겉보기(선수 성장·이적·유소년·재정)가 간접 A와 거의 같지만, 실제 인터랙션(조작 없음)은 직접 계층에 속한다. 사용자는 UI를 보고 **매니저 게임으로 오인**할 가능성이 매우 높다. 이는 7절 리스크 R-C1로 다룬다.

### 4.2 ★ "시드 기반 완전 결정론·재현성"에 대한 냉정한 판정

**결론: 사용자 대상 차별점이 아니다. 개발·QA 자산이며, 마케팅 문구로 전환하려면 별도 투자가 필요하다.**

근거:

1. **보이지 않는다.** 결정론은 "같은 입력에 같은 출력"이라는 속성이며, 사용자는 애초에 같은 입력을 두 번 넣어볼 수단도 동기도 없다. NFR-DT·KPI-3은 전부 **CI/Vitest에서만 관측되는 지표**다.
2. **가장 가까운 성공 사례인 crypto casino의 provably fair조차 조건부다.** provably fair는 server seed 해시 사전 공개 → client seed 사용자 입력 → nonce → `HMAC_SHA256`으로 **사용자가 직접 재계산**할 수 있어야 성립한다. 그리고 이 패턴이 채택된 이유는 명확하다 — **"플레이어와 플랫폼 사이에 소비자 구제 기관이 없는" 실화폐 환경**이기 때문이다. football4는 실화폐가 없으므로 **검증 수요의 원천 자체가 없다**.

1. **Tier 1만으로 2차 릴리스를 개시**하라. 마켓 종류가 아니라 **정산 정확도(KPI-7 오정산 0건)와 마감 무결성(R-11)** 이 2차의 성공 조건이다.
2. **시즌 마켓을 "곁다리"가 아니라 2차의 간판 상품으로 배치**하라 — 유일한 차별 마켓이자 공백 시간의 해답이다.
3. **정확한 스코어의 꼬리 셀렉션 처리 방식을 Task 035(배당)에서 먼저 결정**하라. 이것이 미결이면 Tier 2를 열 수 없다.
4. **`Q-03(배당 표시 형식)`은 Tier 1 오픈 전에 반드시 닫아야 한다.** 국내 사용자는 베트맨의 소수 배당에 익숙하므로 **소수(유럽식) 표기를 기본**으로 권고한다 **[판단]**.

## 7. 리스크 및 시사점

### 7.1 경쟁 관점의 위험 3가지

#### **R-C1 — 카테고리 혼동에 의한 초기 이탈** (영향 높음 / 가능성 높음)

- **내용**: football4의 UI는 선수 상세·클럽 재정·이적 내역 등 매니저 게임과 동일한 요소로 채워진다(FR-UI-005·006·008). 사용자는 Top Eleven·OSM·FM의 경험에서 **"내 팀을 고르고 조작한다"** 를 기대하고 진입한 뒤, **조작 버튼이 없다는 사실을 발견하고 이탈**한다.
- **증거**: 간접 A 계층의 모든 대량 서비스(3억 등록 Top Eleven, 5,000만 OSM, 950만 FPL)가 예외 없이 **사용자 개입을 코어 루프로 삼는다** **[확인]**. 개입 없는 축구 시뮬로 대중적 성공을 거둔 사례는 조사 범위에서 확인되지 않았다.
- **대응 권고**: ① 랜딩·온보딩에서 **"당신은 감독이 아닙니다"를 첫 화면에 명시**한다. ② 개입 욕구를 배팅이 아닌 방식으로도 흡수한다 — **"내 팀 즐겨찾기(follow)"** 는 조작이 아니면서 소유감을 준다(PS-1의 "내가 찍어둔 팀은 어떻게 됐나?" 니즈와 정확히 일치). 1차 릴리스 범위 내 저비용 기능이므로 **FR-UI에 추가 검토를 권고**한다.

#### **R-C2 — 규제 지위 변동** (영향 높음 / 가능성 중간)

- **내용**: 국내에서 **베트맨(스포츠토토) 외의 스포츠 베팅은 전부 불법**이며, 참여자에게 5년 이하 징역 또는 5,000만원 이하 벌금이 규정되어 있다 **[확인]**. football4는 실화폐가 아니므로 현재는 이에 해당하지 않는다. 그러나 **게임산업진흥법 시행령 별표2 제8호에 "스포츠 베팅 게임"을 웹보드게임으로 추가하는 개정이 진행 중**이며, 규제 내용에는 **월 30만원 아이템 구매 한도**, **1회 베팅에 월한도 상당 게임머니의 1/10 초과 사용 금지**, **자동 배팅 방지 의무**가 포함된다 **[확인]**.
- **함의**:
  - 2차 릴리스(무료 포인트 배팅)는 **아이템 판매가 없으므로 웹보드게임 규제의 주 대상은 아닐 가능성이 높다** — 다만 **[추정]** 이며, **법률 검토 없이 확정하지 말 것**.
  - **3차 릴리스의 캐시 충전은 이 규제의 정면 대상이 된다.** 충전을 도입하는 순간 구매한도·베팅한도·자동배팅 금지·연령 확인 의무가 한꺼번에 따라온다.
  - "자동 배팅 방지" 조항은 football4가 향후 **자동 베팅 기능이나 봇 API**를 열 수 없다는 뜻이기도 하다.
- **대응 권고**: ① 3차 캐시 충전은 **법률 검토를 게이트로 걸 것**(6절에서 이미 비권고). ② **UI에서 "배팅/베팅"이라는 용어 사용을 재검토**하고 "예측(prediction)"으로 대체하는 안을 검토하라 — 사용자 인식과 법적 프레이밍 양쪽에서 유리하다 **[판단]**. ③ 마켓·배당 UX가 실화폐 사이트와 지나치게 유사해지지 않도록 주의.

#### **R-C3 — 직접 경쟁의 회전율·화려함 격차** (영향 중간 / 가능성 높음)

- **내용**: 배터 관점에서 football4는 **다음 킥오프까지 평균 37.5분 대기 / 마켓 17종 / 텍스트 중계**인 반면, Betradar VFL은 **40~90초 대기 / 1,000+ 마켓 / 실사 영상**이다 **[확인·계산]**. 배팅 상품으로만 비교하면 **모든 축에서 열위**다.
- **함의**: PS-3(밸류 헌터)를 정면으로 노린 마케팅은 실패할 가능성이 높다. football4의 배팅은 **"배팅 상품"이 아니라 "관전의 부가 레이어"** 로 포지셔닝해야 한다.
- **대응 권고**: ① 5절 Tier 1의 **시즌 마켓 조기 오픈**(공백 시간 해결). ② 3.2절 **라운드 오프셋**. ③ 획득 타겟의 우선순위를 **PS-3 → PS-1/PS-2로 재조정**할 것을 권고한다(페르소나 검증은 팀원 1 문서와 교차 확인 필요).
