**6팀 판정 — 반영한 것과 반영하지 않은 것**:

1. **26~37일차 대기구간·V-01 실측·SP-3(37일차)는 미변경**: 이 구간은 6팀 **자체** 작업(Edge Function CPU 실측)이며 026 도착을 기다리는 게 아니라 026과 별개로 진행할 수 있다 — 026이 39일차로 밀려도 SP-3에서 V-01 결과를 37일차 그대로 전 팀에 공유할 수 있다(팀장 확정, 6팀 최초 판단과 일치). 다만 033 착수는 026 도착(39일차)까지 추가로 기다려야 해 38~39일차에 6팀 자체 작업 없는 대기가 생긴다.
2. **033(구 38~48일차 → 40~50일차) 전량 반영**: Task 033은 H-15(엔진 026)를 착수 조건으로 명시한 유일한 Task라 그대로 2일 순연시켰다. 세부 11개 항목(골격/락/멱등성/상한/백오프/중단감지/헬스기록/시크릿/검증로직/통합테스트/SP-4)은 각각 특정 날짜에 1개씩 배정된 구조라 내부적으로 압축할 여지가 없다(§5 공수 추정의 "8.5인일/11일" 차이는 회의·검토 오버헤드 몫이지, 특정 요일의 여유가 아니다).
