## 1. 팀 소개

### 병합된 원 역할

| 원 팀원 | 역할 | 이 팀으로 이관된 범위 |
|---|---|---|
| **팀원 6** | 데이터 / 밸런싱 | 003, 007, 029, 030, 031, 043 **전량** |
| **팀원 7** 일부 | 배팅 / 배당 도메인 | **035 (몬테카를로 배당 산출)** |

**035를 이 팀에 붙인 근거**

1. 몬테카를로 배당 산출은 결국 **확률 분포 산출**입니다. KPI-4(Brier Score ≤ 0.21) 검증은 이 팀이 이미 수행하는 밸런스 분포 검증(KPI-8)과 같은 작업입니다.
2. **V-02**(배당 시뮬 1회 ≤3.3ms 경량 모델) 실패 시 필요한 것은 "경량 근사 확률 모델"이고, 이는 파라미터·분포 도메인입니다.
3. `MC_N_MATCH`(3,000) / `MC_N_SEASON`(**I-08로 1,500 상향 권고**)은 전부 공통코드이며, 이 팀이 소유합니다.
4. 배당 산출은 엔진(2팀)을 **호출만** 하고 `src/lib/sim/`을 수정하지 않으므로 소유 경로가 겹치지 않습니다.

### 담당 요구사항 ID 범위

FR-PL-001~016, FR-TM-001~010, FR-TR-001~012, FR-EC-001~012, FR-YT-001~006, FR-AW-001~006, FR-AD-011~016, FR-BT-005·006·014, FR-LG-017, NFR-CFG-001~007, NFR-OB-001~005, NFR-DT-006, NFR-PF-004·005·007, NFR-SC-004
결정 전제: **D-16**(절차적 생성), **D-17**(국적 기반 이름), **D-20**(감독 독립 엔티티·최소 이적), **D-21**(재임대 금지), **D-26**(공통코드 시드값)

### 소유 경로 — **이 팀만 커밋합니다**

```
src/lib/config/**            공통코드 36그룹 카탈로그 + 상수 로더

src/lib/data/mock/**         Mock 어댑터 구현체 (인터페이스는 1팀 소유)
```

**유일한 교차 지점**: `src/i18n/messages/{ko,en}/enums.ts` — 4팀 소유 디렉터리 안의 파일 2개만 이 팀이 커밋합니다. 이를 피하기 위해 **13일차에 표시명 목록을 텍스트로 먼저 인계(H-10)** 하고, 4팀이 011에서 파일 골격을 만든 뒤 **23일차 이후** 이 팀이 값만 채우는 순서로 배치했습니다. 같은 날 동시 편집은 없습니다.
