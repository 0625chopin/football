  TypeError가 나 **500이 떨어지던 버그**를 400으로 정정. 회귀 테스트 2건 추가.
- 우리 소유 API 점검 결과 `/api/health`는 GET/DELETE로 입력이 없어 해당 없음. 즉 6팀 소유
